# Experiments for Deep Quadratic Networks

In this repository, we provide the codes, results, artifacts,
and post-processing scripts used to create the results in the following paper:

Abbas Kazemipour, Brett W. Larsen, and Shaul Druckmann.
Avoiding Spurious Local Minima in Deep Quadratic Networks. 
arXiv:2001.00098, 2019. https://arxiv.org/abs/2001.00098
